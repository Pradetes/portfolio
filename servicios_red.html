Contenido de Servicios en Red

En la asignatura de Servicios en Red he adquirido habilidades para la instalación, configuración y gestión de los servicios esenciales que operan en una infraestructura de red. Esto incluye:

Servicios Fundamentales de Infraestructura

  • **Servidor DNS (Domain Name System):** Configuración y gestión de la resolución de nombres de dominio para traducir direcciones IP a nombres de host y viceversa.
  • **Servidor DHCP (Dynamic Host Configuration Protocol):** Implementación de la asignación automática de direcciones IP y otros parámetros de red a los dispositivos clientes.
  • **Servidor NTP (Network Time Protocol):** Sincronización de la hora en los dispositivos de la red para asegurar la consistencia y la correcta operación de los sistemas y logs.

Servicios de Archivos y Acceso Remoto

  • **Servicios de Compartición de Archivos:** Configuración de servidores FTP, Samba (para Windows) y NFS (para Linux/Unix) para el almacenamiento y acceso compartido de archivos en la red.
  • **Acceso Remoto Seguro:** Implementación de servicios como SSH (Secure Shell) para acceso seguro a línea de comandos, y la configuración de VPNs básicas para acceso remoto cifrado a la red.
  • **Servidores de Directorio (LDAP/Active Directory básico):** Gestión centralizada de usuarios, grupos y recursos en entornos de red.

Servicios Web y de Correo Electrónico

  • **Servidor Web (HTTP/HTTPS):** Despliegue y configuración de servidores como Apache o Nginx para alojar páginas web y aplicaciones, incluyendo la gestión de certificados SSL/TLS.
  • **Servicios de Correo Electrónico:** Comprensión y configuración básica de protocolos como SMTP (envío), POP3 e IMAP (recepción) para el funcionamiento del correo electrónico en una red.
  • **Proxy Servers:** Implementación de servidores proxy para el control del tráfico de red, filtrado de contenido o mejora del rendimiento.
