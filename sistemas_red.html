Contenido de Sistemas en Red

En la asignatura de Sistemas en Red, he adquirido conocimientos profundos sobre la administración e implementación de sistemas operativos de servidor y servicios de red en entornos multiusuario y centralizados. Esto incluye:

Administración de Servidores (Windows Server y Linux Server)

  • **Instalación y Configuración:** Despliegue de sistemas operativos de servidor como Windows Server y distribuciones de Linux orientadas a servidor (ej. Ubuntu Server, Debian, CentOS).
  • **Roles y Características del Servidor:** Gestión de roles de servidor (DNS, DHCP, IIS, Apache, etc.) y configuración de sus características para proveer diversos servicios en la red.
  • **Administración Remota:** Uso de herramientas de administración remota (SSH para Linux, Escritorio Remoto para Windows) para gestionar servidores de forma eficiente.

Servicios de Dominio y Directorio

  • **Active Directory (Windows Server):** Implementación y administración de un dominio de Active Directory, incluyendo la creación de controladores de dominio y la gestión de objetos (usuarios, grupos, unidades organizativas).
  • **Políticas de Grupo (GPO):** Configuración de políticas de grupo para aplicar configuraciones de seguridad, software y entorno a usuarios y equipos en un dominio.
  • **Servicios de Directorio en Linux (Conceptos):** Comprensión de la importancia de los servicios de directorio como LDAP en entornos Linux para la autenticación centralizada.

Implementación y Gestión de Servicios Centralizados

  • **Servidores de Archivos y Compartición:** Configuración de almacenamiento compartido y servicios de archivos (ej. DFS en Windows, NFS/Samba en Linux) para acceso centralizado a datos.
  • **Servidores de Impresión:** Despliegue y gestión de servidores de impresión para centralizar la administración de impresoras en la red.
  • **Copia de Seguridad y Restauración:** Estrategias y herramientas para realizar copias de seguridad de datos críticos del servidor y procedimientos de restauración en caso de fallos.
