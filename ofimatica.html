Contenido de Ofimática

En esta asignatura he conseguido conocimientos a nivel profundo sobre ofimática. Esto incluiría lo siguiente:

Procesador de textos

  • Formato Avanzado de Texto y Párrafo: Más allá de la negrita o cursiva, se aprende a controlar el tipo de fuente, tamaño, color, y efectos de texto
  • Maquetación y Estructura del Documento: Esto incluye la configuración de la página (márgenes, orientación vertical u horizontal, tamaño de papel), la inserción y manipulación de objetos como imágenes, tablas, gráficos y formas.
  • Herramientas de Revisión y Colaboración: Se aprende a utilizar eficazmente el corrector ortográfico y gramatical, el diccionario de sinónimos y el contador de palabras.

Hoja de cálculo

  • Uso de Fórmulas y Funciones: Cómo introducir y trabajar con fórmulas para realizar cálculos (suma, resta, multiplicación, división) y el uso de funciones predefinidas
  • Organización y Manipulación de Datos: Habilidades para estructurar datos en celdas, filas y columnas, aplicar formatos (moneda, porcentaje, fecha), ordenar y filtrar información para encontrar datos específicos rápidamente, y usar la validación de datos para asegurar la consistencia.
  • Creación de Gráficos y Visualización de Datos: Cómo transformar conjuntos de datos en representaciones visuales claras y comprensibles (gráficos de barras, líneas, circulares, dispersión, etc.) para identificar tendencias, patrones y hacer presentaciones más impactantes.

Base de datos

  • Diseño y Modelado de Bases de Datos: Aprender a conceptualizar una base de datos, identificar entidades y sus relaciones, y diseñar esquemas de bases de datos relacionales.
  • Lenguaje SQL (Structured Query Language): Dominar las sentencias SQL para interactuar con la base de datos.
  • Administración y Gestión de Datos: Conceptos sobre cómo manejar y mantener una base de datos dentro de un Sistema Gestor de Bases de Datos.

Presentaciones

  • Diseño y Estructura de Diapositivas: Cómo planificar y organizar el contenido de una presentación, utilizando diseños de diapositiva, plantillas y temas para mantener una coherencia visual.
  • Animaciones y Transiciones para Impacto Visual: Dominar el uso de animaciones para dar vida a los elementos individuales dentro de una diapositiva.
  • Herramientas de Presentación y Exposición Efectiva: Aprender a utilizar las vistas del presentador para gestionar tus notas, controlar el tiempo y navegar por la presentación durante la exposición.
