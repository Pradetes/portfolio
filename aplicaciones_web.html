Aplicaciones Web

En la asignatura de Aplicaciones Web he adquirido conocimientos fundamentales para el desarrollo de sitios y aplicaciones interactivas en el entorno web. Esto incluye:

Desarrollo Frontend (Cliente)

  • **HTML (HyperText Markup Language):** Estructura semántica y organización del contenido de las páginas web.
  • **CSS (Cascading Style Sheets):** Estilizado visual, diseño responsive y adaptabilidad para diferentes dispositivos.
  • **JavaScript:** Programación del lado del cliente para añadir interactividad, dinamismo y funcionalidades a la interfaz de usuario.

Desarrollo Backend y Bases de Datos (Servidor)

  • **Lógica del Servidor:** Creación de la parte lógica de la aplicación que maneja la comunicación con la base de datos y el procesamiento de peticiones (ej. con PHP o Node.js).
  • **Bases de Datos Relacionales:** Diseño de esquemas de bases de datos (SQL) y manejo de datos (consultas, inserciones, actualizaciones, eliminaciones) para almacenar y gestionar información.
  • **APIs y Servicios Web:** Comprensión de cómo crear y consumir interfaces de programación de aplicaciones (APIs) para la comunicación entre diferentes sistemas o con el frontend.

Frameworks y Herramientas de Desarrollo

  • **Frameworks Frontend/Backend:** Uso de herramientas y librerías que facilitan y aceleran el desarrollo (ej. Bootstrap para diseño, React/Vue/Angular para JS, Laravel/Express.js para backend).
  • **Control de Versiones (Git):** Manejo de repositorios Git para el control de cambios en el código fuente, colaboración en equipo y gestión de proyectos.
  • **Entornos de Desarrollo:** Configuración y uso de entornos de desarrollo integrados (IDEs) y servidores locales para probar y desplegar aplicaciones.
