Contenido de Servicios Web

En la asignatura de Servicios Web he adquirido conocimientos sobre la infraestructura y los protocolos necesarios para el despliegue, gestión y consumo de servicios y aplicaciones en el entorno de la World Wide Web. Esto incluye:

Servidores Web (HTTP/HTTPS) y Su Configuración

  • **Instalación y Gestión de Servidores Web:** Configuración y administración de servidores HTTP/HTTPS como Apache e Nginx para alojar sitios y aplicaciones web.
  • **Hosts Virtuales y Certificados SSL/TLS:** Creación de hosts virtuales para alojar múltiples sitios en un mismo servidor y la implementación de HTTPS para asegurar las comunicaciones mediante certificados SSL/TLS.
  • **Redirecciones y Reescritura de URLs:** Configuración de reglas de redirección y reescritura para gestionar el acceso a los recursos web de forma eficiente y segura.

Despliegue de Aplicaciones Web y Contenedores

  • **Entornos de Ejecución:** Configuración de entornos para el despliegue de aplicaciones desarrolladas en diferentes lenguajes y frameworks (ej. PHP, Node.js, Python).
  • **Gestión de Aplicaciones Web:** Técnicas para el despliegue, actualización y mantenimiento de aplicaciones web en un servidor, incluyendo la gestión de dependencias y versiones.
  • **Introducción a Contenedores (Docker):** Comprensión de los fundamentos de la virtualización ligera con contenedores para empaquetar aplicaciones y sus dependencias, facilitando el despliegue y la portabilidad.

Consumo y Creación de Web Services (APIs)

  • **Conceptos de Web Services:** Entendimiento de los diferentes tipos de servicios web (ej. SOAP, RESTful APIs) y sus casos de uso en la integración de sistemas.
  • **Consumo de APIs:** Cómo realizar peticiones a APIs existentes, manejar las respuestas (ej. en formato JSON o XML) y utilizarlas para integrar funcionalidades en aplicaciones.
  • **Creación Básica de APIs:** Principios para diseñar y exponer servicios web propios para que otras aplicaciones puedan consumir datos o funcionalidades de nuestra plataforma.
