Contenido de Sistemas Monopuestos

En la asignatura de Sistemas Monopuestos, he adquirido conocimientos esenciales para la instalación, configuración y administración de sistemas operativos tanto en entornos Windows como en distribuciones Linux, centrándonos principalmente en Ubuntu. Esto incluye:

Instalación y Configuración Básica

  • **Instalación de Sistemas Operativos:** Procesos de instalación de Windows (diferentes versiones) y Linux (Ubuntu Server/Desktop), incluyendo particionado de discos y configuraciones de arranque (dual-boot).
  • **Configuración Post-instalación:** Ajustes iniciales del sistema, instalación de drivers, configuración de red básica y personalización del entorno de usuario en ambos sistemas.
  • **Gestión de Usuarios y Permisos:** Creación, modificación y eliminación de usuarios y grupos, y gestión de permisos de archivos y directorios para asegurar la seguridad y privacidad.

Administración de Archivos, Procesos y Software

  • **Sistema de Archivos:** Comprensión de la estructura de directorios en Windows y Linux (ej. `/`, `/home`, `/var`) y manejo de archivos y carpetas (copiar, mover, eliminar, buscar).
  • **Gestión de Procesos y Servicios:** Monitorización y control de procesos en ejecución, y administración de servicios del sistema en ambos sistemas operativos (ej. Administrador de Tareas en Windows, `systemctl` en Linux).
  • **Instalación y Desinstalación de Software:** Uso de gestores de paquetes (APT en Ubuntu) y métodos de instalación en Windows para la gestión eficiente de aplicaciones.
  • **Línea de Comandos:** Dominio de comandos esenciales en la terminal de Linux (Bash) y en el Símbolo del Sistema/PowerShell de Windows para automatizar tareas y solucionar problemas.

Configuración de Red y Seguridad Básica

  • **Configuración de Red:** Asignación de direcciones IP estáticas/dinámicas, configuración de DNS y gateway en interfaces de red de ambos sistemas operativos.
  • **Firewall Local:** Configuración básica del firewall de Windows (Windows Defender Firewall) y el firewall de Linux (UFW) para controlar el tráfico de red entrante y saliente.
  • **Actualizaciones y Mantenimiento:** Importancia de mantener el sistema operativo y el software actualizados, y técnicas de mantenimiento preventivo para optimizar el rendimiento y la seguridad.
